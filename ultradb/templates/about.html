About Page

This Web Application will allow a logged-in user to interact with the database. (Certain features are available without being logged in.)

The Web App provides an interface for viewing and updating information pertaining to:

  • Job Sites with:
    • List of Areas within a Site
    • List of Rooms within an Area
    • Any Colorsheets associated with an area
  • Painting Projects, past, present and future with:
    • Project details (location, dates, status, type, quote)
    • List of areas / rooms included in project scope
    • Labour breakdown for project
  • Employee Time Tracking. Logged in users can:
    • Enter their time against a particular project
    • View a summary of their logged hours

Report Generation

The Web App can also generate reports:

  • List of rooms that are overdue for painting
  • List of rooms "As Needed" areas sorted by last painted date

Future Features:

  • Interface for updating date of last paint
    • Alternatively, auto update date_last_paint for rooms in project upon completion
  • Material Breakdown for project
  • List of areas / rooms painted, sorted by most recent
  • Areas touched within a specified filtered date range
  • Excel Report of all room data for chosen site
