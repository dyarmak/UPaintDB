About Page

This Web Application will allow a logged-in user to interact with the database.

The Web App provides an interface for viewing and updating information pertaining to:

Note: Certain features are only available to logged in users.

  • List of Job Sites with:
    • List of Areas within a Job Site
    • List of Rooms within an Area
    • Show any Colorsheets associated with an area
  • Painting Projects, past, present and future with:
    • Project details (location, dates, status, type, quote)
    • List of areas / rooms included in project scope
    • Labour breakdown for project
    • Example of above
    • Search by BM_id and name
    • Manually edit room details
    • Upon project completion, any rooms associated with that project have their "Date Last Painted" field updated
    • Each room has a Painting History associated with it.
  • Employee Time Tracking. A logged in users can:
    • Enter their time against a particular project
    • View a summary of their logged hours

Report Generation

The Web App can also generate the following reports:

  • List of rooms that are "Overdue" for painting
  • List of rooms that are "As Needed", sorted by last painted date

FUTURE Reports:

  • List of rooms painted in a selected date range, default to most recently painted
  • Labour hours by date range
  • Labour hours by project
  • Excel Report of all room data for chosen site (For Craig)

In future, this will be easily exportable/downloadable to an excel file

Future Features:

  • Material tracking for projects
  • PO and WO number fields for projects
  • A dashboard that graphs various metrics (Rev/Exp by site, hours per week, project profitability, etc)
  • Admin panel to change user permission levels

Future Improvements

  • Better sorting of Sites / Areas / Rooms / Projects
  • Filters (radio buttons) on view project route, by type (SM,AN,CT), by status (inprogress, complete, invoiced)
  • Weekly automated hours reporting to owner / bookeeper
  • Scheduled Database Backups
